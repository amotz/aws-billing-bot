AWS billing notification bot for slack, on AWS Lambda function.

## Usage

````````
SLACK_API_TOKEN = 'xxxx-xxxxxx-xxxxxx'
SLACK_CHANNEL = '#xxxxx'
````````

Set any trigger on Lambda (e.g. CloudWatch Events:cron).
